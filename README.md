# Melbourne Housing Price Prediction with PyTorch

Learning PyTorch by building a regression model for house price prediction.

## Setup
```bash
# Install dependencies
pip install -r requirements.txt

# Get dataset from Kaggle and place in data/raw/

# Train model
cd src
python train.py
```

## Project Structure

- `src/` - Source code
- `data/` - Dataset storage
- `models/` - Saved models
- `results/` - Training outputs
- `notebooks/` - Jupyter notebooks
